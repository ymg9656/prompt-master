# Prompts 디렉터리 가이드

- 역할별 하위 폴더에 모델별 템플릿을 배치합니다.
- 각 템플릿은 **시스템 메시지 → 사용자 프롬프트 템플릿 → 출력 형식 → 점검 목록 → 모델별 팁** 순서로 구성됩니다.
- 추가 역할이 필요하면 새 폴더를 만들고 네 가지 모델(ChatGPT, Claude, Cursor, 제미나이) 파일을 동일한 규칙으로 추가하세요.
- 사용자 프롬프트 템플릿의 `{{ }}` 플레이스홀더에 프로젝트 정보를 채워 바로 복사해 사용할 수 있습니다.
- 팀마다 선호하는 표/포맷이 다를 수 있으니, 출력 형식 섹션을 참고해 세부 서식을 지정하세요.
