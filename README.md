# Prompt Master

목적에 맞는 프롬프트를 역할과 모델별로 구조화해 보관하는 저장소입니다. 확장성을 고려해 디렉터리를 구성했으며, 각 문서는 바로 복사해 사용할 수 있는 기본 템플릿을 제공합니다. 모든 작업은 `develop` 브랜치에서 진행합니다.

## 구조
- `prompts/` 역할별 프롬프트 템플릿 모음
  - `pm/` PM 전용 템플릿
  - `planner/` 기획자 전용 템플릿
  - `design/` 디자인 전용 템플릿
  - `frontend/` 프론트엔드 전용 템플릿
  - `backend/` 백엔드 전용 템플릿

각 역할 폴더에는 ChatGPT, Claude, Cursor, 제미나이 전용 템플릿이 포함되어 있으며, 필요 시 동일한 구조로 폴더와 파일을 추가해 확장할 수 있습니다.

## 템플릿 사용법
1. 필요한 역할/모델 파일을 열고 `{{ }}` 플레이스홀더에 프로젝트 정보를 채웁니다.
2. **시스템 메시지 → 사용자 프롬프트 → 출력 형식 → 점검 목록 → 모델별 팁** 순으로 확인해 복사/붙여넣기 합니다.
3. 표나 코드 블록 등 원하는 출력 형식을 명시하면 더 일관된 응답을 얻을 수 있습니다.
